While making this website there were a few things that I wanted to make sure I added to it. This artical is about the first which is called prettyprint. Prettyprint is made by google. It is a colection of js, css, html, and a bit of perl. With this library we can add pieces of code to our website for readers to see. The library will than add syntax highlighjting to the code to make it more readable. Here is an example of a few lines of JavaScript.

        var a = 5;
        var b = 10;
        function add(a, b) {
          return a + b;
        }
        console.log(add(a, b));

Useing this library we can include pieces of code in articles and tutorials which wil make the code easier to understand for readers. Adding prettyprint to a website is very easy. Simply include this line in your webside. 

        <script src="https://cdn.jsdelivr.net/gh/google/code-prettify@master/loader/run_prettify.js?skin=sunburst"></script>

Now, if you ever want a piece of code in your website include a <pre> tag with the class "prettyprint". Everything in the pre tags will be syntax highlighted. To learn more about this visit the github page here. Also, you can change the skin based on how you want your website to look. We're useing this skin because it fits our website.
